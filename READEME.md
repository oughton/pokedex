# Pokedex

Submission from Joel Oughton.

## Minimum requirements

- .NET Core 3.1 runtime (https://dotnet.microsoft.com/download/dotnet/3.1/runtime)

## Run instructions

- Open terminal 
- Run `cd .\Pokedex\Api`
- Run `dotnet run`
- Execute HTTP requests (e.g., http://localhost:5000/pokemon/mewtwo)
